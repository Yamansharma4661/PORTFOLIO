My Projects

VISUAL READ ASSIST

The "Visual Read Assist" project is an innovative, wearable device designed to empower visually impaired individuals by providing real-time auditory assistance. It features modern spectacles equipped with a camera module connected to a Raspberry Pi, housed in a protective case. The device captures text from books or other sources using Optical Character Recognition (OCR) technology and converts it into natural-sounding audio through Text-to-Speech (TTS). The audio output is delivered via headphones or speakers, while a push-button control ensures ease of operation. This low-cost, portable system eliminates the need for expensive Braille machines, offering accessibility and independence in education, navigation, and daily tasks. The user-friendly interface supports additional functionalities like storing images and digital backups, making it a versatile tool that significantly improves the quality of life for visually impaired individuals.

PROJECT CERBERUS

Project Cerberus is a multi-phase robotics solution for disaster management, addressing challenges like inaccessible zones, delays in rescue operations, and risks to human responders. It features a five-phase system comprising aerial drones for infrared-based surveillance and mapping, land robots for navigating debris and locating victims, drone-guided navigation to optimize rescue paths, on-site medical assistance through land robots, and nursing robots for delivering medical supplies in confined spaces. The technology integrates thermal imaging, LiDAR, GPS, and advanced sensors for precise victim detection and communication. By enabling faster rescue operations, enhancing safety, and providing real-time data, Project Cerberus revolutionizes disaster management and ensures efficient, adaptable, and continuous rescue missions.

Hybrid Power Bank with Storage

The "Hybrid Power Bank with Storage" is a revolutionary device that combines portable charging with high-speed data storage. It features a compact, durable design with multiple charging ports, an LED display for battery status, and integrated SSD or HDD storage options ranging from 512GB to several TBs. The device supports wireless connectivity via Wi-Fi and Bluetooth, enabling remote file access and data encryption for security. Ideal for travelers, content creators, and professionals, it simplifies carrying multiple gadgets by offering dual functionality in a single portable solution.
